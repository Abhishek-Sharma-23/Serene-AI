# 🧠 SereneAI Therapist

SereneAI Therapist is an AI-powered mental wellness assistant designed to provide supportive conversations and promote emotional well-being.
The project is built using Python and aims to simulate a calm and supportive AI therapist that can interact with users in a friendly and helpful way.

---

## 🚀 Features

* 🤖 AI-powered conversational assistant
* 💬 Mental wellness support and guidance
* 🧩 Modular project structure
* ⚡ Lightweight Python implementation
* 🧠 Designed for learning and experimentation with AI chat systems

---

## 📂 Project Structure

```
SereneAI-Therapist
│
├── app.py               # Main application file
├── requirements.txt     # Python dependencies
├── README.md            # Project documentation
│
├── assets/              # Static assets
└── utils/               # Helper modules and utilities
```

---

## ⚙️ Installation

Clone the repository:

```bash
git clone https://github.com/Abhishek-Sharma-23/Serene-AI.git
cd Serene-AI
```

Install dependencies:

```bash
pip install -r requirements.txt
```

Run the application:

```bash
python app.py
```
